# HLMagicTools
### 开源 iOS framework工具库



### Build This Code
> 编译开发环境 Xcode 7.2.1

1. 安装 [最新版Xcode](https://developer.apple.com/download/)
2. Clone the repository:

	```shell
	git clone https://github.com/HarwordLiu/HLMagicTools.git
	```
3. 打开HLMagicTools_App.xcodeproj
4. 编译Framework

> 1.切换project 编译 com + b
